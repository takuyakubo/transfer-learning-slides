# 転移学習（Transfer Learning）スライド

機械学習プロフェッショナルシリーズの転移学習に関する勉強会資料です。
このリポジトリでは勉強会で使ったスライドを公開しています。

## 内容
- 第1章: 機械学習から転移学習へ
- 第2章: 転移学習の基礎
- 第3章: メイン適応の理論

2025年3月31日作成
2025年5月20日更新
